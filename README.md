# Prueba Técnica Frontend – Logika

## Descripción
Aplicación web desarrollada en React que implementa autenticación mediante token JWT, consumo de APIs protegidas y un dashboard administrativo para la gestión de acciones.

## Tecnologías utilizadas
- React
- TypeScript
- Vike
- Fetch API
- LocalStorage

## Instalación y ejecución
```bash
npm install
npm run dev

## Limitaciones conocidas

- El endpoint `POST /api/v1/actions/admin-add` retorna 404 en el entorno de desarrollo.
- La funcionalidad de creación fue implementada y validada a nivel de frontend, incluyendo validaciones y manejo de errores.
- El comportamiento fue documentado y manejado de forma controlada.
